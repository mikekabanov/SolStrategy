# ⚡ Quick Start: 5-Minute GitHub Deployment

## TL;DR Version - Deploy in 5 Minutes

### Step 1: Create Repository (2 min)
```
→ Go to https://github.com/new
→ Name: SolStrategy
→ Public
→ Create
```

### Step 2: Upload Files (2 min)
```bash
cd c:\data\SolStrategy
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/USERNAME/SolStrategy.git
git branch -M main
git push -u origin main
```

### Step 3: Enable GitHub Pages (1 min)
```
→ Settings → Pages
→ Deploy from branch → main → /
→ Save
```

✅ **LIVE AT**: `https://USERNAME.github.io/SolStrategy`
